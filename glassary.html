Estrutura HTML

!DOCTYPE HTML

  avisar o navegador para seguir as especificações do documento.

HTML

  É a Linguagem de Marcação de Hipertexto, já o elemento <html> </html> é a raiz dos outros elementos HTML.

head

  É a parte do código não visivel no browser e que tem a função de configurar a pagina, sendo escrita da seguinte maneira: “<head> </head>”.

body

  É a parte do código que apresenta o conteudo da pagina, sendo escrito assim: “<body> </body>”.
